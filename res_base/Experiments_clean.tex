
We consider the one-dimensional heat equation with Dirichlet, Neumann, and periodic boundary conditions, discretized on a two-qubit register. Unless otherwise stated, the numerical target is the exact discretized solution vector $u(T)=e^{-AT}u(0)$ at total evolution time $T=1$, with oscillator truncation $n_{\mathrm{fock}}=64$ and first-order Trotterization using $n_{\mathrm{trotter}}=100$ steps.

For the numerical study we fix a four-dimensional semidiscrete system encoded on a two-qubit register, with $M=4$, $\alpha=1$, $h=1$, $T=1$, and initial state $u(0)=\ket{01}$. Since $\alpha/h^2=1$ in all three cases, the exact semidiscrete generators used in the simulations are

\subsection{Oracle-baseline parameter selection}

We first use refined injection-based sweeps to select the kernel parameters $r_{\mathrm{target}}$, $r^\prime$, and $\beta$. These sweeps should be interpreted as oracle-baseline studies: the CV resource state is loaded ideally, so the resulting sensitivity curves isolate kernel quality rather than gate-level state-preparation error. The corresponding exact-fidelity curves are shown in Fig.~\ref{fig:clean_sensitivity_exact}. These plots are envelope plots over the remaining swept parameters.

Across all three boundary conditions, the best oracle-baseline operating points lie in a narrow high-squeezing region. The resulting exact fidelities are already above $0.9989$ in all three cases.

\subsection{Empirical relation to the theoretical bounds}

The state-preparation theorem, Theorem~\ref{thm: state-prep}, is asymptotic and contains unknown constants $\mathcal K$ and $\mathcal M$, so the present experiments can only test qualitative consistency rather than a calibrated numerical tightness ratio. Within the refined injection sweeps, increasing the oscillator coefficient cutoff from $n_{\mathrm{coeff}}=24$ to $48$ improves the best exact fidelity in all three cases: Dirichlet increases from $0.9956816$ to $0.9989611$, Neumann from $0.9995599$ to $0.9997155$, and periodic from $0.9992957$ to $0.9997249$. This behavior is consistent with Theorem~\ref{thm: state-prep}, which predicts improved approximation as the truncated oscillator resources are increased. The same data also place the optimum in a large-squeezing regime, with both $r_{\mathrm{target}}$ and $r^\prime$ near $4$ to $8$, which is qualitatively consistent with the squeezing dependence in the theorem, although not by itself a proof of numerical tightness.

For the hybrid evolution, Eq.~\eqref{eq:heat-trotter-error} predicts first-order scaling in the number of product-formula steps. Using the existing Dirichlet-focused $n_{\mathrm{trotter}}$ sweep in \texttt{results\_clean\_refine}, and comparing against the truncated CV reference rather than the exact PDE solution, the best observed relative errors are $3.0835\times 10^{-3}$ at $n_{\mathrm{trotter}}=8$, $1.5428\times 10^{-3}$ at $n_{\mathrm{trotter}}=16$, and $7.7171\times 10^{-4}$ at $n_{\mathrm{trotter}}=32$. These values track a normalized $C/n$ guide line very closely, providing a representative empirical check of the expected first-order behavior. At present, this quantitative tightness test is available only for the existing low-to-moderate Dirichlet sweep region, rather than for the refined high-fidelity operating points or for all three boundary conditions.

Among the presently available practical runs, the strongest result is the Neumann case, where the matched exact-fidelity gap is only 0.020. By contrast, the weakest currently available practical case is Dirichlet, where the loss is dominated by imperfect CV state preparation rather than by the hybrid evolution.

The presently available successful SNAP$+$D runs were obtained at nearby high-fidelity kernel points identified before the final refined oracle sweep, rather than by a full re-optimization over the updated oracle-baseline optima. \subsection{Physical interpretation and resource costs}

The high-fidelity oracle-baseline points occur at large values of $r_{\mathrm{target}}$ and $r^\prime$. In the CV-DV LCHS construction, the ancillary oscillator enters through the qumode sandwich $\langle \phi | e^{-it(\hat{x}\otimes L + I\otimes H)} | \psi \rangle$, so these high-squeezing values indicate that the CV resource state must realize a broad enough quadrature-space kernel profile to cover the relevant spectral window of $A$. The gain in exact PDE fidelity therefore comes with a smaller postselection probability. At the same time, the truncated-model fidelities remain essentially unity, which shows that once the oscillator seed state is fixed, the hybrid evolution itself tracks the finite-Fock target very accurately.

This distinction is reflected directly in the practical runs. For the Neumann problem, the current low-depth SNAP$+$D ansatz already prepares the oracle state accurately enough to preserve nearly all of the matched injection fidelity. For Dirichlet and periodic boundary conditions, however, the end-to-end fidelity loss is dominated by oscillator state preparation: the executed circuit remains close to the exact truncated map induced by the prepared SNAP$+$D seed, but the prepared seed itself is still too far from the ideal oracle state.

For Givens synthesis, the current clean analysis provides an analytic exact-synthesis baseline together with JC-pulse and qubit-rotation counts. In the present runs, Givens matches the injection fidelity to numerical precision for all three boundary conditions while reporting 47 JC pulses and 47 qubit rotations.
