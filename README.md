# Workspace Guide

Code and retained data for the CV-DV LCHS benchmarks. All numbers in the
revised submission are generated from the artifact tree
`results_revision_v2/` (see `results_revision_v2/README.md` for the full
per-table provenance and reproduction commands); the operating point is one
shared parameter set, (r, r', beta, N) = (1.6, 0.25, 0.5, 32) at
N_Fock = 64, for all three boundary conditions
(`results_revision/selected_params_v2.json`).

## Law-Eberly state preparation (method notes)

The `law_eberly` state-preparation method implements the recursive
Law-Eberly protocol for arbitrary finite oscillator states
([Law and Eberly, PRL 76, 1055](https://doi.org/10.1103/PhysRevLett.76.1055);
[Hofheinz et al., Nature 459, 546](https://doi.org/10.1038/nature08005)).
For a target

```math
|g\rangle \otimes |\psi\rangle
  = |g\rangle \otimes \sum_{n=0}^{N} c_n |n\rangle ,
```

the compiler solves the time-reversed problem by repeatedly removing the
highest occupied Fock amplitude. Each inverse step first applies a
Jaynes-Cummings exchange that couples

```math
|e,n-1\rangle \leftrightarrow |g,n\rangle ,
```

then applies a global auxiliary-qubit rotation that removes the remaining
qubit excitation. The preparation circuit is the adjoint of that
unpreparation sequence. At the Bosonic Qiskit level, the exchange pulse is

```math
S_n(\alpha,\phi)=
\exp\!\left[-i\frac{\alpha}{\sqrt n}
\left(e^{i\phi}\sigma_-a^\dagger+e^{-i\phi}\sigma_+a\right)\right],
```

implemented by `cv_jc(alpha / sqrt(n), phi, qumode, qubit)`, and the qubit
rotation is

```math
R_n(\theta,\varphi)=
I_{\rm osc}\otimes
\exp\!\left[-\frac{i\theta}{2}
\left(\cos\varphi\,X+\sin\varphi\,Y\right)\right],
```

implemented by `r(theta, phi, qubit)` on the auxiliary qubit. This is an
ideal circuit-level Law-Eberly model; it does not include calibrated flux
pulses, detuning trajectories, leakage, or decoherence.
